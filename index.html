Tudo Sobre HTML:

A linguagem HTML foi criada por Tim Berners-Lee por volta de 1989, inicialmente não como uma especificação, mas como uma ferramenta de comunicação entre pesquisadores. Uma das primeiras aplicações da web era a disponibilização de currículos online de membros de universidades e instituições de pesquisa nos Estados Unidos. Enquanto as regras de sintaxe do HTML se desenvolviam ao longo do tempo, essa linguagem desempenhava um papel fundamental na evolução da World Wide Web.

Embora as regras de sintaxe do HTML 1.0 fossem notavelmente flexíveis, foi apenas com a versão 2.0 no início da década de 1990 que se estabeleceu uma especificação mais formal. Isso desencadeou a "guerra dos navegadores", e em 1995, o HTML 3.0 obrigou os desenvolvedores da web a criar duas páginas distintas, uma compatível com o Microsoft Internet Explorer e outra com o Netscape Navigator. Essas duas versões apresentavam avaliações muito distintas e introduziam novos recursos de forma independente. A criação do World Wide Web Consortium (W3C) em 1996 trouxe algum grau de ordem, e essa comunidade lançou a versão 3.5 do HTML.

Em 1997, surgiu a linguagem HTML com o objetivo de formalizar sua definição. No ano seguinte, houve uma reforma completa do edifício A. O HTML 4.0, desenvolvido pelo W3C, foi lançado em 1998. A ideia central era separar a apresentação visual da estrutura da página. Isso deu origem ao CSS (Cascading Style Sheet), que ficou responsável pelo design da página, enquanto o HTML era reservado para definir a estrutura da página.

No decorrer de 1999, várias revisões levaram à criação do HTML 4.01, que é o padrão recomendado pelo W3C em uso atualmente. Finalmente, em 2000, o XHTML 1.0 foi introduzido. Trata-se, essencialmente, de uma versão do HTML escrita em eXtensible Markup Language (XML), indicada pelo pequeno "x" no início. O XHTML posteriormente evoluiu com a formalização do XHTML 1.1.

Desde a virada do século, o W3C continuou a lançar novas atualizações, bugs e muito mais. No entanto, ainda existem novas recomendações para HTML 5.0 que não demorarão muito.
